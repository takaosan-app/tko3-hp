# tko3servs

tko3servs は、takaosan.app から派生した小さなWebサービス群です。

## コンセプト

takaosan.app を個人開発の母屋とし、そこで生まれたAPIや軽量ツールを、単独で使いやすい形に切り出して公開する。

大きなプラットフォームのように見せるより、日々の開発や実務で「これだけ使いたい」と思える小さな道具を丁寧に並べる。郵便番号、住所変換、地理情報など、身近で実用的な処理を中心に育てていく。

## 表現方針

- takaosan.app の派生サービスであることを、トップとフッターで明示する。
- tko3servs 自体は、APIとツールの実用サイトとして読みやすさを優先する。
- 色や余白は takaosan.app と近い温度感にしつつ、サービスサイトとして少し機能的に見せる。
- 英語は補助的なラベルに留め、日本語で用途がわかるようにする。
- 「小さく切り出した道具」「takaosan.app からの枝分かれ」という文脈を保つ。

## ページ構成

- トップページ: tko3servs が takaosan.app から派生したサービス群であることを説明する。
- Origin: takaosan.app との関係性を説明する。
- Services: 現在公開しているAPIとツールを一覧化する。
- Apps: 実際に試せる軽量ツールを並べる。
- 各APIページ: 実用性を優先しつつ、冒頭に派生サービスである文脈を短く入れる。
