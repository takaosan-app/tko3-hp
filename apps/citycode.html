API ドキュメント

市区町村コード API

住所文字列から総務省の市区町村コード（5桁）を返すREST API。郵便番号APIと組み合わせることで、住所から行政コードを一括取得できます。

概要

ベースURL

api.tko3servs.com

認証

不要（無料）

レスポンス形式

JSON

データソース：総務省「全国地方公共団体コード」（soumu.go.jp）。コードは団体コード6桁のうち先頭5桁（チェックデジット除く）を使用します。

エンドポイント

GET /citycode 
https://api.tko3servs.com/citycode?q={住所}

パラメータ

パラメータ 必須 説明
q 必須 住所文字列（都道府県から入力推奨。例: 静岡県藤枝市稲川）

レスポンス例

200 OK
application/json 
{
  "query":     "静岡県藤枝市稲川",
  "city_code": "22214",
  "pref":      "静岡県",
  "city":      "藤枝市",
  "pref_kana": "シズオカケン",
  "city_kana": "フジエダシ"
}
フィールド 型 説明

仕様メモ

政令指定都市の区

住所に「区」が含まれる場合（例: 北海道札幌市中央区）、市ではなく区のコードを返します。住所に区名がなく市名だけの場合は市のコードを返します。

東京23区

東京の特別区（千代田区・新宿区など）は政令指定都市ではなく独立した市区町村として扱われます。東京都新宿区 のように入力してください。

コードの桁数

総務省の団体コードは6桁ですが、このAPIは末尾のチェックデジットを除いた5桁を返します（業界標準として広く利用されている形式）。

エラー

400

Bad Request

qパラメータが空または未指定です。

404

Not Found

住所から都道府県・市区町村を特定できませんでした。
