API ドキュメント

近隣スポット API

緯度・経度を指定すると、近くの公園・施設などのスポットを距離順で返すREST API。OpenStreetMapデータを使用。認証不要・無料でご利用いただけます。

概要

ベースURL

api.tko3servs.com

認証

不要（無料）

レスポンス形式

JSON

エンドポイント

GET /spots 
https://api.tko3servs.com/spots?lat={緯度}&lon={経度}

パラメータ

パラメータ 必須 デフォルト 説明
lat 必須 — 緯度（−90〜90）
lon 必須 — 経度（−180〜180）
radius 任意 500 検索半径（m）。100〜5000
limit 任意 5 最大件数。1〜20

レスポンス例

200 OK
application/json 
{
  "lat":          35.6895,

フィールド 型 説明

エラー

400

Bad Request

lat・lonパラメータが不正、または範囲外の値です。

422

Unprocessable Entity

必須パラメータ（lat・lon）が指定されていません。
